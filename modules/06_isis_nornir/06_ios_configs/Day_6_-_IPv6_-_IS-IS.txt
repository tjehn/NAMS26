=====================================================
Configure BGP with the ISP and redistribute BGP-OSPF
=====================================================

-----
R1
-----

router bgp 111
 neighbor 2111:1234:1111::9 remote-as 1000
 address-family ipv6
  neighbor 2111:1234:1111::9 activate
  redistribute ospf 1 
!
ipv6 router ospf 1
 redistribute bgp 111

-----
R9
-----

router bgp 1000
 neighbor 2111:1234:1111::1 remote-as 111
 address-family ipv6
  neighbor 2111:1234:1111::1 activate

-----
R2
-----

router bgp 222

=========================================================================================
Configure NAT64 to allow the IPv4 devices to be seen as IPv6 devices on the IPv6 network 
=========================================================================================

---------------------------------------------------------------
Enable NAT64 on the appropriate interface on the NAT64 Device
---------------------------------------------------------------

----
R4
----

Interface E 0/0
 nat64 enable
!
Interface E 0/1
 nat64 enable
!
Interface E 0/2
 nat64 enable

---------------------------------------------------------------
Allocating an IPv6 address space for NAT64 and advertising it
---------------------------------------------------------------

----
R4
----

nat64 prefix stateful 2000:1234:ABCD:0400::/64
!
ipv6 route 2000:1234:ABCD:0400::/64 null 0
!
ipv6 router ospf 1
 redistribute static

----
R1
----

router bgp 111
 address-family ipv6
  redistribute ospf 1 match internal external

---------------------------------------------------------------------------------------------------Configuring the Static NAT64 to translate the internal IPv4 devices as IPv6 devices on the outside
---------------------------------------------------------------------------------------------------

----
R4
----

nat64 v4v6 static 6.1.1.1 2000:1234:ABCD:0400::1
nat64 v4v6 static 6.1.1.2 2000:1234:ABCD:0400::2
nat64 v4v6 static 6.1.1.3 2000:1234:ABCD:0400::3

---------------------------------------------------------------------------------------------------Configuring the Dynamic PAT for public IPv6 addresses to a pool of Internal IPv4 addresses
---------------------------------------------------------------------------------------------------

----
R4
----

ipv6 access-list NATLIST
 permit ip 2000::/3 any
!
nat64 v4 pool V4POOL 10.10.10.1 10.10.10.2
!
nat64 v6v4 list NATLIST pool V4POOL overload
